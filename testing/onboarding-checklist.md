# Onboarding User Flow Checklist

- [ ] 1. Navigate to the app homepage / landing page.
- [ ] 2. Click "Sign Up" or "Create Account".
- [ ] 3. Enter a valid email address.
- [ ] 4. Enter a valid password (meets requirements, if any).
- [ ] 5. Confirm password if prompted.
- [ ] 6. Submit the sign-up form.
- [ ] 7. Verify email (if required): check inbox and click verification link.
- [ ] 8. Log in with the newly created credentials.
- [ ] 9. Complete profile setup (name, username, avatar/initial preferences).
- [ ] 10. Save profile.
- [ ] 11. Confirm redirection to the dashboard/home feed.
- [ ] 12. Verify profile data appears correctly on the dashboard.
- [ ] 13. Optional: confirm account settings show the new profile info.
- [ ] 14. Log out and repeat steps 2–13 if necessary (edge cases, invalid inputs).

Acceptance criteria: A new user can sign up, verify (if needed), log in, complete profile, and land on the dashboard seeing their profile data.
